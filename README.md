# Portfolio Intelligence

A Streamlit-based project portfolio risk and resource intelligence app designed to help teams prioritize projects, detect delivery risk, identify resource overload, and support scenario planning.

## Overview

Portfolio Intelligence is a decision-support application for project portfolio management. It brings together project overview, risk intelligence, resource intelligence, scenario planning, and executive briefing views in one interactive interface.

The app is designed to simulate how a PMO or project leadership team can monitor portfolio health and make better prioritization decisions using analytics.

## Features

- Portfolio overview dashboard
- Risk intelligence analysis
- Resource utilization and overload detection
- Scenario planning for decision support
- Executive briefing view
- Multi-page Streamlit interface
- Modular Python codebase for analytics and recommendations

## How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/abhitatamiya05-pixel/portfolio-intelligence.git
cd portfolio-intelligence
```

2. Create and activate a virtual environment:

```bash
python -m venv venv
source venv/bin/activate
```

On Windows:

```bash
venv\Scripts\activate
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

4. Run the app:

```bash
streamlit run app.py
```
